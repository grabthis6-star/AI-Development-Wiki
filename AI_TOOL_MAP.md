# AI와 도구 지도

목적에 맞는 AI 서비스, 내부 기능과 개발 도구를 발견하고 조합하기 위한 카탈로그입니다. 기능과 요금제는 바뀔 수 있으므로 각 공식 링크에서 최신 정보를 확인하세요.

## OpenAI

### 💬 ChatGPT
- 종류: AI 서비스
- 한 줄 설명: 아이디어를 대화로 구체화하고 문서·이미지·자료를 함께 다루는 범용 작업 공간
- 추천 상황: 무엇을 만들지 정리하거나, 초안을 만들고, 파일과 이미지를 보며 피드백할 때
- 필요 없는 상황: 프로젝트 파일을 직접 계속 수정·실행해야 할 때는 Codex가 더 적합합니다.
- 사용자가 준비할 것: ChatGPT 계정과 만들고 싶은 결과에 대한 설명
- 공식 안내: https://chatgpt.com/
- 요청문:

> 내 아이디어를 프로그램의 사용자 흐름, 핵심 기능 3개와 첫 완료 기준으로 정리해줘.

### 📂 ChatGPT Projects
- 종류: AI 내부 기능
- 한 줄 설명: 관련 채팅·파일·지침을 한 프로젝트 안에서 계속 사용하는 공간
- 추천 상황: 같은 프로그램을 여러 채팅에 걸쳐 오래 만들거나 참고 문서를 계속 사용해야 할 때
- 필요 없는 상황: 한 번만 묻는 짧은 질문에는 일반 채팅으로 충분합니다.
- 사용자가 준비할 것: 프로젝트 목적, 참고 파일, 반복해서 지킬 작업 원칙
- 공식 안내: https://chatgpt.com/
- 요청문:

> 이 프로젝트의 파일과 지침을 기준으로 답하고, 이전 결정과 충돌하면 먼저 알려줘.

### 🛠️ Codex
- 종류: 개발 에이전트
- 한 줄 설명: 프로젝트 폴더를 읽고 코드 수정·실행·검증까지 맡는 AI 개발 담당자
- 추천 상황: 실제 프로그램 기능 추가, 버그 수정, GitHub 반영과 브라우저 검증이 필요할 때
- 필요 없는 상황: 아이디어만 자유롭게 대화하거나 단순 문서 초안만 필요할 때
- 사용자가 준비할 것: 프로젝트 폴더, 컨텍스트 문서, 이번 작업, 유지할 기능, 완료 기준
- 공식 안내: https://developers.openai.com/codex/
- 요청문:

> 현재 프로젝트와 AGENTS.md를 먼저 읽고 이번 기능만 구현해. 기존 기능을 보호하고 실제 동작까지 검증해.

### 🗺️ Codex Plan mode
- 종류: Codex 기능
- 한 줄 설명: 코드를 바꾸기 전에 AI가 상황을 확인하고 작업 계획을 세우는 모드
- 추천 상황: 요구가 모호하거나 여러 파일·단계가 얽힌 기능을 시작할 때
- 필요 없는 상황: 문구 하나처럼 작고 명확한 수정에는 바로 작업해도 됩니다.
- 사용자가 준비할 것: 목표, 현재 문제, 절대 바꾸면 안 되는 부분
- 공식 안내: https://learn.chatgpt.com/docs/codex
- 요청문:

> 아직 수정하지 말고 현재 프로젝트를 확인한 뒤 목표·변경 범위·검증 방법을 계획해.

### 📜 AGENTS.md
- 종류: Codex 기능
- 한 줄 설명: Codex가 프로젝트에서 반복해서 지켜야 할 규칙을 저장하는 문서
- 추천 상황: 같은 설명이나 실수가 반복되고, 실행·검증 방법을 모든 작업에 적용해야 할 때
- 필요 없는 상황: 이번 한 번만 필요한 요청은 채팅에 적는 편이 낫습니다.
- 사용자가 준비할 것: 폴더 구조, 실행 방법, 유지 원칙, 완료 기준
- 공식 안내: https://learn.chatgpt.com/docs/agent-configuration/agents-md
- 요청문:

> 최근 반복된 실수를 정리하고 짧고 실용적인 AGENTS.md 규칙으로 제안해.

### 🧩 Skills
- 종류: Codex 기능
- 한 줄 설명: 자주 반복하는 전문 작업 절차를 AI가 다시 사용할 수 있게 묶는 기능
- 추천 상황: 문서 제작·검증·배포처럼 같은 순서의 작업을 여러 프로젝트에서 반복할 때
- 필요 없는 상황: 한 번뿐인 작업이나 아직 절차가 안정되지 않은 작업
- 사용자가 준비할 것: 검증된 작업 순서, 입력 자료, 완료 조건
- 공식 안내: https://learn.chatgpt.com/docs/codex
- 요청문:

> 이 작업은 반복될 예정이야. 먼저 안정적인 절차를 정리하고 재사용 가능한 Skill로 만들 가치가 있는지 판단해.

### 🔌 Plugins와 MCP

- 추천 상황: AI가 프로젝트 밖의 최신 파일·업무 데이터·서비스를 직접 읽거나 작업해야 할 때
- 필요 없는 상황: 파일을 직접 첨부하는 것으로 충분하거나 연결 권한이 불필요할 때
- 사용자가 준비할 것: 연결할 서비스, 필요한 최소 권한, 사용자 승인
- 공식 안내: https://learn.chatgpt.com/docs/plugins
- 요청문:

> [서비스]를 연결해 [작업]을 하고 싶어. 필요한 최소 권한과 안전한 연결 방법을 먼저 설명해.

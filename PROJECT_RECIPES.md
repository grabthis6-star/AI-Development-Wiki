# 프로젝트 만들기 레시피

아이디어를 사용자 흐름과 확인 가능한 첫 버전으로 바꾸는 예시입니다. 완성 기능을 한 번에 만들지 않고 가장 작은 결과부터 시작합니다.

## 프로젝트 예시

### 🗣️ 영어 표현 학습 도구
궁금한 영어 문장을 붙여 넣고 뜻·표현·예문을 정리합니다.
- 사용자: 영상이나 글에서 발견한 영어 표현을 실제로 써보고 싶은 학습자
- 해결할 문제: 모르는 표현을 검색해도 뜻만 확인하고 다시 복습하지 못합니다.
- 사용 흐름: 영어 문장 붙여넣기 → 뜻과 핵심 표현 확인 → 새 예문 확인 → 복습 목록에 저장
- 첫 버전: 영어 문장 입력 / 뜻·핵심 표현·예문 표시 / 결과 복사
- 나중에 추가: 로그인 / 자동 복습 알림 / 음성 평가 / 학습 통계
- 완료 기준: 영어 문장 하나를 입력하면 뜻·표현·예문이 한 화면에 표시됩니다.
- AI 시작문:

> 영어 학습자가 영상에서 들은 문장을 붙여 넣으면 뜻, 핵심 표현, 쉬운 예문을 한 화면에서 확인하는 웹 도구를 만들어. 첫 버전은 문장 입력, 분석 결과, 결과 복사만 구현하고 로그인과 학습 통계는 제외해.

### 🎬 영상 제작 보조 도구
영상 아이디어를 장면·대사·필요 자료로 나누어 편집 전에 정리합니다.
- 사용자: 영상 편집을 시작하기 전에 전체 구성을 빠르게 잡고 싶은 제작자
- 해결할 문제: 아이디어는 있지만 편집 프로그램을 열면 장면 구성부터 다시 고민하게 됩니다.
- 사용 흐름: 영상 아이디어 입력 → 장면 구성 생성 → 대사·자료 정리 → 순서와 시간 확인
- 첫 버전: 아이디어 입력 / 장면 목록 생성 / 장면별 메모 수정
- 나중에 추가: 영상 자동 생성 / 음성 합성 / 자막 자동 배치 / 편집 프로그램 연동
- 완료 기준: 한 줄 아이디어를 입력하면 수정 가능한 장면 목록이 생성됩니다.
- AI 시작문:

> 영상 제작자가 한 줄 아이디어를 입력하면 장면 순서, 화면 설명, 대사와 예상 시간을 수정 가능한 목록으로 받는 웹 도구를 만들어. 첫 버전은 장면 생성과 수정만 구현하고 실제 영상 생성은 제외해.

### 📋 업무 정리 도구
회의 메모를 할 일·담당자·기한·확인 질문으로 구조화합니다.
- 사용자: 회의 후 해야 할 일을 빠뜨리지 않고 정리하고 싶은 개인이나 작은 팀
- 해결할 문제: 회의 메모는 남지만 누가 무엇을 언제까지 해야 하는지 다시 정리해야 합니다.
- 사용 흐름: 회의 메모 붙여넣기 → 할 일 후보 확인 → 담당자·기한 수정 → 목록 복사
- 첫 버전: 메모 입력 / 할 일 목록 변환 / 사용자 수정과 복사
- 나중에 추가: 자동 이메일 발송 / 캘린더 등록 / 팀 권한 / 진행률 대시보드
- 완료 기준: 회의 메모를 입력하면 수정 가능한 할 일 목록이 표시됩니다.
- AI 시작문:

> 사용자가 회의 메모를 붙여 넣으면 할 일, 담당자, 기한과 확인 질문으로 나누어 보여주는 웹 도구를 만들어. 사용자가 결과를 수정하고 복사할 수 있게 하고 자동 이메일과 캘린더 등록은 첫 버전에서 제외해.
